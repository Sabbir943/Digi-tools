## 📌 Project Name
**A-6 Digital Tools Buying Website**

---

## 📝 Description
This is a modern and responsive digital tools marketplace where users can explore premium tools, view detailed features, and add products to a cart. The project provides a smooth user experience with interactive UI, dynamic cart functionality, and real-time notifications.

---

## 🛠️ Technologies Used
- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🌼 DaisyUI  
- 📜 JavaScript (ES6+)  
- 🔔 React-Toastify  
- 📦 JSON Data  

---

## ✨ Key Features

### 🛒 1. Smart Cart System
Users can add products to the cart, remove individual items, and see the total product count update instantly in the navbar.

### 🔄 2. Product & Cart Toggle
Users can easily switch between product view and cart view. By default, the product section is displayed for a better browsing experience.

### 🔔 3. Real-time Notifications
All actions like adding to cart, removing items, and checkout are shown using toast notifications for better user interaction.
